## Описание
Random Quote Generator - это графическое приложение на Python, которое генерирует случайные цитаты из предопределённого списка. Приложение позволяет сохранять историю просмотренных цитат, фильтровать их по автору и теме, а также добавлять новые цитаты в коллекцию.

## Основные функции

### 1. Генерация случайных цитат
- Нажмите кнопку "Сгенерировать цитату" для получения случайной цитаты
- Цитата отображается в верхней части окна с указанием автора и темы

### 2. Фильтрация цитат
- **По автору**: выберите автора из выпадающего списка для генерации цитат только этого автора
- **По теме**: выберите тему для генерации цитат только по выбранной теме
- Кнопка "Очистить фильтр" сбрасывает все фильтры

### 3. История цитат
- Все сгенерированные цитаты сохраняются в истории
- История отображается в виде таблицы с полями: цитата, автор, тема, время генерации
- Возможность очистить всю историю
- Автоматическое сохранение истории в файл `quote_history.json`

### 4. Добавление новых цитат
- Заполните форму в нижней части окна: текст цитаты, автор, тема
- Нажмите "Добавить цитату" для пополнения коллекции
- Проверка на пустые поля: все поля обязательны для заполнения

### Требования
- Python 3.6 или выше

2. Откройте терминал/командную строку
3. Перейдите в директорию с файлом
4. Выполните команду:
